# GOSSIP – Messaging Application (Java)

## Description
GOSSIP is a desktop-based messaging application developed using Java.
The application allows users to register, log in, send messages,
manage conversations, and interact through a structured user interface.

The project follows the MVC architecture to ensure clean code organization,
readability, and maintainability.

## Technologies Used
- Java
- JavaFX
- MVC Architecture
- Object-Oriented Programming (OOP)

## Features
- User registration and login
- One-to-one messaging
- Message management
- Group messaging
- User settings and profile control
- Controller-based navigation between views

## Project Structure
- **Controller Layer** – Handles user interactions and application flow
- **Model Layer** – Manages data structures and business logic
- **View Layer** – JavaFX-based graphical user interface

## Project Files
- `Controller.java` – Main application controller
- `RegisterController.java` – User registration and authentication
- `MesajSayfasiController.java` – Messaging and conversation management
- `AyarlarController.java` – User settings management
- `GroupManager.java` – Group and participant management

## How to Run
1. Make sure Java is installed.
2. Open the project in an IDE (IntelliJ IDEA or Eclipse).
3. Run the main application file.
